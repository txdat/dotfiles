# Workflow: /create-issue — Create Standalone Issue

Use this for issues not tied to a plan: bugs, spikes, discussions, or quick tasks.

1. **Collect Data**: title, description, labels, milestone.
2. **Execute**:
   ```bash
   gh issue create --title "..." --body "..." [--label "..."] [--milestone "..."]
   ```
3. **Report**: Print the issue URL.
